Flux 5

iOS 11.0 - 13.3.x untethered jailbreak

To use Flux, you must install it to your device using ReProvision, Cydia Impactor or another sideloading method (like Xcode).

Using ReProvision is currently recommended, as Cydia Impactor has been down for a while.

If you are currently jailbroken with another jailbreak, you can install it using Filza and AppSync

NOTE: Flux should work properly, but in case it does not, we are not responsible for any loss of property or time.



IMPORTANT: Do not update your device before running Flux. iOS indexes its files after an update, and this causes this jailbreak to fail.
